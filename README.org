* rae-dux
Endgame wireless split ergo keeb with low-profile switches and keycaps

[[https://i.imgur.com/UK9RJ01.jpg]]

** Features
1. Minimal design - no diodes, LEDs, OLEDs
2. Ultra low-profile and transportable
